### 1、机器学习常用场景
- 聚类场景：人群划分和产品种类划分。
- 分类场景：广告投放预测和网站用户点击预测。
- 回归场景：降雨量预测、商品购买量预测和股票成交额预测。
- 文本分析场景：新闻的标签提取、文本自动分类和文本关键信息抽取。
- 关系图算法场景：社交网络关系挖掘和金融风险控制。
- 模式识别：语音识别、图像识别和手写字识别。
